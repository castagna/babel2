
  BABEL2
  ======

  What is this?
  -------------

  Babel2 is a rewritten (but mainly copied) version 2 of SIMILE's Babel 
  to convert between various formats through RDF semantic models.

  This is just a an experimental prototype.

  For the original SIMILE Babel look here:
  http://service.simile-widgets.org/babel/
  http://simile-widgets.googlecode.com/svn/babel/trunk/


  Credits
